Welcome

Marvel QA is a student project organized by the Language Technology group of the Computer Sience Department in TU Darmstadt. The goal of Marvel QA is to develop a web-based application that utilizes the IBM Watson Question answering system. This project was developed by Wei-Hung Hsu, your name and your name. This project is currently a WIP.

Introduction

Ask Watson uses IBM Watson Question answering system to answer questions regarding the Marvel Cinematic Universe. The data set that was used for the training was extracted from the Marvel Cinematic Universe Wikia. To enter a question, simply type in your question into the text box below and press enter.
