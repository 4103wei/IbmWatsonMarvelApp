Welcome

Marvel QA is a student project organized by the Language Technology group of the Computer Sience Department in TU Darmstadt. The goal of Marvel QA is to develop a web-based application that utilizes the IBM Watson Question answering system. This project was developed by Wei-Hung Hsu, Erich Wittenbeck and Khine Htwe.

Introduction

Ask Watson uses IBM Watson Question answering system to answer questions regarding the Marvel Cinematic Universe. The data set that was used for the training was extracted from the Marvel Cinematic Universe Wikia. To enter a question, simply type in a question into the text box and press enter. Add Questions allows users to add their own questions to the Marvel Quiz. The quiz utilizes IBM Watson Question answering system to generate possible answers. The answers are further specified through entity recognition.

How it works


The front-end was developed using Angular2. "AngularJS is a complete JavaScript-based open-source front-end web application framework mainly maintained by Google and by a community of individuals and corporations to address many of the challenges encountered in developing single-page applications." 1


For the back-end, which is responsible for the data processing, and the access of data-base, we used Tomcat. "Apache Tomcat, often referred to as Tomcat, is an open-source web server developed by the Apache Software Foundation (ASF). Tomcat implements several Java EE specifications including Java Servlet, JavaServer Pages (JSP), Java EL, and WebSocket, and provides a "pure Java" HTTP web server environment in which Java code can run."2
